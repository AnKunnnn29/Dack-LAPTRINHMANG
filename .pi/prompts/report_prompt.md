# Report Prompt

Write a short, clear Markdown report for Network Recon + Risk Profiler.

Mandatory rules:

- Do not include exploit steps.
- Do not include payloads.
- Do not include brute-force, bypass, or real attack guidance.
- Only provide defensive observations and recommendations.
- Include a MITRE ATT&CK Mapping section based on the `mitre_mapping` field.
- Include an ML Risk Model section that summarizes the model and important features.
- Use a style suitable for a student final project presentation.

Required sections:

1. Target
2. Recon Summary
3. Risk Level
4. ML Risk Model
5. Findings
6. MITRE ATT&CK Mapping
7. Recommendations
8. Conclusion
